== Description ==

This plugin is easy to set category ranking.
If you view the post,Ranking is on the same category.

This plugin:

* Easy setting ranking widget on same category
* Frontpage ranking is able to choose

Please contact me if you want additional setting.

== Installation ==

1. Upload the `simple-category-index` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setting [Appearance]->[Widgets]->[Simple Category Ranking]
